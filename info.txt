
Para correr solo desde clasificar_paginas.py, no borres data/trabajo ni storage/tmp/pages.
1. Limpiar BD del lote
SQL
UPDATE documentos_paginas
SET estado = 'separado',
    tipo_detectado = NULL,
    serie = NULL,
    numero = NULL,
    ruc_emisor = NULL,
    razon_social_emisor = NULL,
    orden_servicio = NULL,
    orden_compra = NULL,
    clave_documental = NULL,
    requiere_qr = FALSE,
    qr_procesado = FALSE,
    qr_raw = NULL,
    qr_error = NULL,
    banco_abreviatura = NULL,
    codigo_operacion = NULL
WHERE cliente_abreviatura = 'BBTEC'
  AND anio = 2026
  AND mes = 1;

DELETE FROM documentos_agrupados
WHERE cliente_abreviatura = 'BBTEC'
  AND anio = 2026
  AND mes = 1
  AND origen = 'fragmentos';
2. Limpiar archivos generados
Bash
rm -rf data/salida/2026/BBTEC/01/provisional
rm -rf data/salida/2026/BBTEC/01/con_oc
find data/salida/2026/BBTEC/01/sin_oc -type f -name "04-*.pdf" -delete
No borres los archivos de facturas 1 página en sin_oc que empiezan con BBTEC FACTURA....
3. Ejecutar de nuevo
Bash
python app/clasificar_paginas.py --year 2026 --cliente BBTEC --month 1
python app/procesar_qr_pendientes.py --year 2026 --cliente BBTEC --month 1
python app/agrupar_fragmentos.py --year 2026 --cliente BBTEC --month 1
python app/distribuir_agrupados.py --year 2026 --cliente BBTEC --month 1
Con eso reutilizas las páginas ya separadas y corriges la clasificación.
