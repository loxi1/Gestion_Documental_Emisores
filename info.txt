Sí. Esos 27 no se distribuirán porque están en:
Plain text
revision_manual_qr
y además no tienen agrupado:
Plain text
agrupado_id = NULL
ruta_final = NULL
Lo que pasa es esto:
Plain text
tipo_detectado = guia_remision
clave_documental = OTRO|asiento|pagina
estado = revision_manual_qr
O sea: el sistema sospechó que eran guías, pero no pudo completar serie/número/RUC por QR, entonces las dejó fuera.
Antes de distribuir tienes 2 caminos:
Opción segura para cerrar el mes
Si ya revisaste y aceptas que queden como OTRO, márcalas como clasificado:
SQL
UPDATE documentos_paginas
SET estado = 'clasificado',
    tipo_detectado = 'otro',
    requiere_qr = FALSE,
    qr_procesado = TRUE,
    qr_error = NULL
WHERE cliente_abreviatura='BBTEC'
  AND anio=2026
  AND mes=4
  AND estado='revision_manual_qr'
  AND clave_documental LIKE 'OTRO|%';
Luego:
Bash
python app/agrupar_fragmentos.py --year 2026 --cliente BBTEC --month 4
python app/distribuir_agrupados.py --year 2026 --cliente BBTEC --month 4
python app/reporte_integridad_lote.py --year 2026 --cliente BBTEC --month 4
Opción más exacta
Editar una por una:
Bash
python app/editar_pagina_manual.py --id 936
