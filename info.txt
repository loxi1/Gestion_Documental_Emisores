Correcto: sí tienen asiento contable en el nombre, pero extraer_desde_nombre() no lo guardó porque tu formato no calzaba con factura. Para OTRO, debemos extraer solo el asiento con regex.
En marcar_extraido_otro.py, agrega esta función:
Python
import re
Python
def extraer_asiento(nombre: str | None) -> str:
    m = re.search(r"\b(04-\d{4})\b", nombre or "")
    return m.group(1) if m else "SIN_ASIENTO"
